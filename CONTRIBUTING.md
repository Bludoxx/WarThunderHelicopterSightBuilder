# Contributing

Bug reports and focused fixes are welcome.

Before opening an issue, check the compatibility table in the README and test
the latest release. A useful bug report includes:

- the builder version;
- the helicopter and view mode;
- whether the helicopter has rocket CCIP or an HMD HUD;
- exact steps to reproduce the problem;
- a screenshot and saved design when relevant.

For code changes:

1. Keep the change focused.
2. Run `run_tests.ps1`.
3. Build with `build_release.ps1`.
4. Explain what changed and how it was tested in the pull request.

Do not include War Thunder account data, game logs containing personal
information, or unrelated extracted game files.
