# Support

Start with the compatibility table in the README.

The full cockpit sight and live range require a non-HMD helicopter with rocket
CCIP, such as the Ka-50, Mi-24, or Mi-28. Helicopters without rocket CCIP and
HMD-HUD helicopters can still use the custom sight in third person, but the
range value does not update there.

For builder bugs, open a GitHub issue and include the version, helicopter, view
mode, reproduction steps, and a screenshot or saved design.

For War Thunder account, launcher, anti-cheat, or game-support questions,
contact Gaijin Support. This repository is an independent community project.
