# Code of Conduct

Keep project discussions useful and respectful.

- Discuss the work, not the person.
- Do not harass, threaten, discriminate against, or deliberately embarrass
  other contributors.
- Do not post private information, account details, or credentials.
- Give people room to correct mistakes.
- Keep bug reports and pull requests on topic.

The repository owner may edit, hide, or remove material that breaks these
rules, and may block repeated or serious abuse.

If a conduct problem involves sensitive information, contact the repository
owner privately through GitHub instead of opening a public issue.
